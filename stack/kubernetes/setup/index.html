# 部署 Kubernetes

目前，Kubernetes 支持在多种环境下使用，包括本地主机（Fedora）、云服务（Google GAE、AWS 等）。

你可以使用以下几种方式部署 Kubernetes：

  • kubeadm
  • docker-desktop
  • k3s
